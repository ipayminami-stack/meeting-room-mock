# 部屋管理機能の実装計画

## 目的
承認者（approver）ロールに以下の機能を追加：
1. 予約状況参照ダッシュボード（ObserverDashboard）
2. 部屋の追加・編集・削除機能

## 実装内容

### 1. RoomManagementコンポーネント作成
**ファイル**: `src/components/features/room-management.tsx`

**機能**:
- 部屋一覧表示
- 部屋の追加モーダル
- 部屋の編集モーダル
- 部屋の削除確認ダイアログ

**入力フィールド**:
- 部屋名（必須）
- 定員（必須、数値）
- 設備（複数選択可能）
- フロア（任意）

### 2. ApproverDashboard更新
**ファイル**: `src/app/dashboard/page.tsx`

**変更内容**:
- approverロールにObserverDashboardを追加
- RoomManagementセクションを追加

**画面構成**:
```
┌─ 承認者ダッシュボード ─────────┐
│ 1. 承認待ち予約リスト          │
│ 2. 部屋管理                    │
│ 3. 予約状況参照                │
└────────────────────────────────┘
```

### 3. データ管理
**状態管理**:
- `useState`で部屋リストを管理
- MOCK_ROOMSを初期値として使用
- 追加・編集・削除時に状態を更新
