2. 「ログイン」ボタンをクリック

## 予約の申請

### 手順

1. **カレンダーから日付を選択**
   - 空き状況カレンダーで予約したい日付をクリック

2. **予約詳細を入力**
   - 会議室を選択
   - 日時を選択
   - 開始時刻と終了時刻を設定
   - 利用目的を入力
   - 参加者情報を入力

3. **外部来訪者の登録（必要な場合）**
   - 「外部来訪者あり」にチェック
   - 来訪者の会社名、氏名、メールアドレスを入力
   - 複数の来訪者がいる場合は「来訪者を追加」ボタンで追加

4. **申請を送信**
   - 「予約を申請」ボタンをクリック

### 注意事項
- 予約は承認者の承認が必要です
- 承認されるまで予約は確定しません

## 予約の確認

### 自分の予約一覧

**今後の予約**
- ダッシュボードの「📅 今後の予約」セクションに表示
- ステータス別に色分け表示
  - 🟠 申請中（承認待ち）
  - 🔵 承認済
  - 🟡 変更申請中

**過去の予約**
- 「📁 過去の予約」をクリックして展開
- 過去の予約履歴を確認可能

### カレンダー表示
- 月間カレンダーで予約状況を確認
- 各予約には以下の情報が表示されます：
  - 時間帯（例：10:00-11:00）
  - 会議室（コラボ/大/小）
  - 利用目的

## 予約の変更

### 承認済み予約の変更手順

1. **変更したい予約を選択**
   - 「今後の予約」から承認済みの予約を探す
   - 「変更」ボタンをクリック

2. **変更内容を入力**
   - 日時、会議室、利用目的、参加者などを変更
   - 変更理由を入力（承認が必要な場合）

3. **変更を申請**
   - 「変更を申請」ボタンをクリック

### 変更の種類

**承認が必要な変更:**
- 日時の変更
- 会議室の変更
- 外部来訪者の追加

**即座に反映される変更:**
- 外部来訪者の削減
- 参加者の削減

### 変更申請の確認
- 変更申請中の予約には「🔄 変更申請」バッジが表示されます
- 変更内容の詳細が表示されます

## 予約の取り消し

### 手順

1. **取り消したい予約を選択**
   - 「今後の予約」から該当の予約を探す

2. **取り消しボタンをクリック**
   - 「申請を取り下げる」（申請中の場合）
   - 「予約を取り消す」（承認済みの場合）

3. **確認**
   - 取り消しは即座に反映されます

## よくある質問

**Q: 予約はいつまでに申請すればいいですか？**
A: 利用日の前日までに申請してください。

**Q: 予約の承認にはどのくらい時間がかかりますか？**
A: 承認者の確認次第ですが、通常1営業日以内に承認されます。

**Q: 外部来訪者の情報は必須ですか？**
A: 外部来訪者がいる場合は必須です。セキュリティ管理のため正確な情報を入力してください。

**Q: 予約を変更したいのですが、承認が必要ですか？**
A: 日時や会議室の変更、外部来訪者の追加は承認が必要です。参加者や来訪者の削減は即座に反映されます。
